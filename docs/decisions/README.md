# Decision Log

Records of human decisions made during each AI-DLC loop (see
[README.md](../../README.md#7-skill-capture--reuse)): choosing an approach, setting a rule, or
redirecting a plan. Recurring or valuable decisions are distilled into skills stored in this
repo's own [`.claude/skills/`](../../.claude/skills/) (candidates awaiting promotion live in
[`docs/knowledge-asset/published/`](../knowledge-asset/published/)) — not in the retired private
`mekhal/claudeskill` repo (see
[2026-07-13-skills-storage-move-in-repo.md](2026-07-13-skills-storage-move-in-repo.md)).
